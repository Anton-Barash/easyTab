# Установка PWA на Android

## Шаг 1: Откройте приложение в Chrome

Откройте приложение в браузере Chrome на вашем Android-устройстве:
```
http://ваш-локальный-ip:5173
```
или через туннель (ngrok), если хотите открыть из интернета.

## Шаг 2: Добавьте на главный экран

1. Нажмите на **меню** (три точки) в правом верхнем углу Chrome
2. Выберите **"Добавить на главный экран"** или **"Установить приложение"**
3. Подтвердите установку

## Шаг 3: Запустите приложение

После установки на главном экране появится иконка **easyTab**. Нажмите на неё для запуска.

## Особенности установленного PWA

- ✅ Приложение открывается **без адресной строки** браузера (fullscreen)
- ✅ Работает ** автономно** (офлайн режим для базовых функций)
- ✅ Появляется в **списке приложений** Android
- ✅ Поддерживает **push-уведомления** (если настроены)
- ✅ Быстрый запуск с главного экрана

## Если кнопка "Установить" не появляется

1. Убедитесь, что вы используете **Chrome** (не Firefox или другой браузер)
2. Откройте приложение и прокрутите страницу **вниз**
3. Вы должны увидеть баннер "Добавить на главный экран"
4. Или нажмите меню → "Добавить на главный экран"

## Публикация для других пользователей

Чтобы другие пользователи могли установить приложение:

1. Опубликуйте приложение на хостинге (Vercel, Netlify, GitHub Pages)
2. Убедитесь, что HTTPS включён (обязательно для PWA)
3. Другие пользователи смогут установить приложение так же, как описано выше

## Требования

- Android 7.0 или выше
- Chrome 73 или выше
- HTTPS-соединение (для публичной версии)
