=== ЗАДАНИЕ ДЛЯ TRAE SOLO BUILDER ===
Проект: Приложение для заполнения отчётов по шаблонам Excel с фото/видео
Стек: Vue 3 + Vite + Pinia + Vue Router + js-xlsx + html2canvas
Платформа: Веб (PWA), адаптив под ПК и мобильные (Android/iOS)
Язык интерфейса: Русский

---
1. ОПИСАНИЕ ПРОЕКТА
Приложение позволяет:
- выбирать папки (шаблоны, фото, отчёты) — отдельно для ПК и телефона
- работать с Excel-шаблонами вопросов (встроенный пример + пользовательские)
- заполнять ответы, добавлять пометки «Внимание!»
- делать фото/видео или выбирать из галереи → авто-переименование + сохранение в выбранную папку
- экспортировать отчёт: Excel (ссылки на медиа) + HTML (превью + кликабельные ссылки)
- сохранять настройки в файл config.json

---
2. ОСНОВНЫЕ ЭКРАНЫ (VIEWS)
2.1 StartView.vue (стартовый экран)
- Превью встроенного шаблона Excel (миниатюра + название)
- Кнопки:
  • Создать новый отчёт (по шаблону)
  • Открыть существующие отчёты
  • Настройки (выбор папок)
- Инструкция: «Выберите шаблон или продолжите работу»

2.2 TemplateSelect.vue
- Список шаблонов из выбранной папки + встроенный пример
- Предпросмотр первых строк Excel
- Кнопка: «Использовать шаблон»

2.3 FormFillView.vue (основной экран заполнения)
- Список вопросов (из шаблона) — компонент QuestionCard.vue
- Каждая карточка:
  • Текст вопроса
  • Поле для ответа (текст)
  • Поле «Внимание!» (необязательное)
  • Блок MediaBlock.vue: фото/видео (съёмка/выбор), превью, удалить
- Навигация: Назад / Далее / Сохранить черновик

2.4 ReportView.vue
- Список сохранённых отчётов (Excel + HTML)
- Превью HTML-отчёта
- Кнопки: Открыть / Скачать / Удалить

2.5 SettingsView.vue
- Выбор папок:
  • Папка с шаблонами
  • Папка для фото/видео
  • Папка для готовых отчётов
- Авто-определение платформы (ПК/телефон) — разная логика выбора папок
- Кнопки: Сохранить настройки / Сбросить

---
3. КОМПОНЕНТЫ
3.1 QuestionCard.vue — вопрос + ответ + пометка + медиа
3.2 MediaBlock.vue — съёмка/выбор фото/видео, превью, удаление
3.3 FolderSelect.vue — универсальный выбор папки (ПК/телефон)
3.4 NavControl.vue — кнопки навигации
3.5 SettingsPanel.vue — блок настроек папок

---
4. ХРАНИЛИЩА (PINIA STORES)
4.1 configStore.js
- Пути к папкам
- Последний выбранный шаблон
- Платформа (ПК/телефон)
- Сохранение/загрузка из config.json

4.2 formStore.js
- Список вопросов (из шаблона)
- Ответы, пометки «Внимание!»
- Привязанные фото/видео (пути, имена)
- Черновики

4.3 reportStore.js
- Список сохранённых отчётов (пути, даты)

---
5. UTILS (УТИЛИТЫ)
5.1 excelLogic.js
- Чтение Excel-шаблона (.xlsx)
- Экспорт готового отчёта: ответы + пометки + ссылки на медиа
- Сохранение в выбранную папку

5.2 htmlReport.js
- Генерация HTML-отчёта:
  • Все вопросы, ответы, пометки
  • Кликабельные ссылки на фото/видео
  • Адаптивный дизайн
- Сохранение в выбранную папку

5.3 mediaManager.js
- Съёмка фото/видео (мобильные)
- Выбор из галереи (ПК/телефон)
- Авто-переименование: `YYYYMMDD-HHMMSS-questionN.jpg`
- Сохранение в выбранную папку
- Проверка форматов: jpg, png, mp4

5.4 folderCompat.js
- Определение платформы (ПК/телефон)
- Логика выбора папки:
  • ПК: полный доступ к файловой системе
  • Телефон: доступ к внешней памяти (Android) / документы (iOS)

5.5 configIO.js
- Чтение/запись config.json
- Хранение в локальном хранилище (браузер) или в файле (ПК)

---
6. ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ
6.1 Папки
- Все пути сохраняются в config.json
- При первом запуске — предложить выбрать папки
- Поддержка русских названий папок/файлов

6.2 Шаблоны
- Встроенный пример: `public/default-template.xlsx`
- Чтение шаблонов: первая колонка — вопросы, вторая — типы (текст/медиа)
- Сохранение заполненного шаблона как новый

6.3 Фото/Видео
- Поддержка камеры (мобильные)
- Выбор нескольких файлов
- Авто-переименование без конфликтов
- Превью в приложении
- Удаление медиа из отчёта (без удаления файла)

6.4 Экспорт
- Excel: отдельная строка на вопрос, столбцы: вопрос, ответ, внимание, путь к медиа
- HTML: один файл, все данные + ссылки, открывается в любом браузере
- Оба файла сохраняются в выбранную папку для отчётов

6.5 Адаптив
- Полная поддержка мобильных устройств (вертикальная ориентация)
- ПК: горизонтальная ориентация, боковая панель с вопросами
